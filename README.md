## 프로젝트 배경/목표
1. 바이오 기반 신약 개발 과정에 대해 이해한다.
2. 신약 개발에 AI를 어떻게 사용할 수 있는지에 대해 이해한다.
3. 바이오 데이터를 이해하고 바이오 데이터에서 활용할 수 있는 전처리 방식에 대해서 학습한다.
4. 어떤 AI 모델링을 통해서 바이오 데이터를 신약 개발에 활용할 수 있을지에 대해서 학습한다.
